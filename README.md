# CognitiveOS

A full-stack AI system with:
- React frontend
- Node.js backend
- Python AI service

## Architecture

client/ → React UI  
server/ → Node API  
python-service/ → AI processing

## Setup

### Install dependencies
npm install

### Run server
cd server
npm install
npm start

### Run client
cd client
npm install
npm start

### Run python service
cd python-service
pip install -r requirements.txt
python app.py
